# 女专办品牌规范

本文是「女专办」的品牌唯一事实源：品牌标几何、色彩、各场景的落地参数，以及改动时要同步的文件清单。v1.10.0 起品牌资产全面矢量化（此前是 15 张 PNG 位图）。

## 名称

- **中文名**：女专办（「女专红」致敬江西师大女子专科学校历史；「办」呼应灵感来源「酱紫办」——把教务的事**办**了）。
- **英文/包名**：Nvzhuanban，包名 `cn.jxnu.nvzhuanban`（历史既定，不随品牌迭代变更）。
- 副标语：**江西师范大学 · 掌上教务**（登录页在用，见 `strings.xml` 的 `login_subtitle`）。

## 品牌标「开卷有成」

翻开的双层书页（学业、教务）+ 上扬对勾（事项办结）。全部由三条圆头描边构成，无填充面，任意缩放不糊。

### 母版几何（1024×1024 视口）

| 元素 | pathData | 描边宽 |
|---|---|---|
| 书页上层 | `M176,592 Q380,608 512,672 Q644,608 848,592` | 62 |
| 书页下层 | `M152,714 Q370,732 512,800 Q654,732 872,714` | 62 |
| 对勾 | `M344,438 L486,570 L736,250` | 100 |

描边一律 `round` 线帽 + `round` 拐角。内容包围盒约 x∈[121,903]、y∈[200,831]（含描边），几何中心即视口中心。

### 同步清单（改几何必须五处一起改）

1. `app/src/main/res/drawable/ic_brand_logo.xml` — 应用内裸标（白描边，使用处 tint）
2. `app/src/main/res/drawable/ic_launcher_foreground.xml` — 桌面图标前景（同时是 monochrome 层）
3. `app/src/main/res/drawable/ic_splash_logo.xml` — 启动页图标
4. `docs/assets/brand/logo.svg` — 图标形态（README 头图）
5. `docs/assets/brand/logo-mark.svg` — 裸标形态（文档/印刷）

## 色彩

| 角色 | 值 | 用途 |
|---|---|---|
| **女专红**（品牌锚点色） | `#A91D34` | 启动页背景（`res/values/colors.xml` 的 `brand_nvzhuan_red`）、M3 浅色主题 primary、白底文档上的裸标 |
| 图标渐变·亮端 | `#B72440` | 桌面图标背景左上 |
| 图标渐变·暗端 | `#8F1226` | 桌面图标背景右下（对角线渐变，中值≈女专红） |
| 深色主题品牌色 | `#FFB3B6` | M3 深色 primary；深色背景上的裸标用它或纯白 |

- 完整 Material 3 色板（由女专红作 seed 派生）在 `ui/theme/Color.kt`，那里是应用内配色的事实源。
- 白标落在女专红上的对比度 ≈ 7.2:1，落在渐变暗端更高；任何新场景保持「白标只放在女专红或更深的底上」。
- 用户开启 Material You 动态取色后应用内主题色随壁纸变化，这是刻意设计——品牌色只锁桌面图标与启动页两个场景。

## 场景参数

| 场景 | 资源 | 关键参数 |
|---|---|---|
| 桌面自适应图标 | `mipmap-anydpi/ic_launcher{,_round}.xml` | 背景=红渐变矢量；前景=白标缩放 0.68（最远端点半径 281 < 312 安全区半径，1024 视口坐标）；monochrome 复用前景 |
| 启动页 | `values{,-night}/themes.xml` | 背景=女专红纯色（两种模式同值）；图标=`@drawable/ic_splash_logo`（内容缩放 0.68，落在系统 2/3 圆形可视域内）；浅色/深色**共用同一资源** |
| 登录页 BrandHeader | `LoginScreen.kt` | 88dp 圆角 24dp、底=primary；标 60dp、tint=**onPrimary**（别写死白色——深色主题 primary 是浅粉，白标会没进去） |
| 「关于」弹窗 | `ProfileScreen.kt` `AboutDialog` | AlertDialog `icon` 槽，44dp，tint=primary |
| README 头图 | `docs/assets/brand/logo.svg` | 128px 居中 |

## 使用规则

1. 品牌标是**描边图形**，缩放时按整体缩放，不要单独调描边宽。
2. 红底场景一律白标；白底场景红标（`#A91D34`）；深色 surface 场景用 `#FFB3B6`。禁止在红底上放红标或引入第三方配色。
3. 桌面图标前景改动后，务必核对新几何最远端点仍在自适应图标 66/108 安全区内（换算：1024 视口坐标下缩放后半径 ≤ 312）。
4. 需要重新视觉验证时，把三条 path 塞进任意 HTML `<svg>`，用无头 Chrome 截图对比圆形/方圆遮罩、48px 小尺寸、themed icon 单色、深浅色八个场景（本次升级的做法，无需装任何设计工具）。
